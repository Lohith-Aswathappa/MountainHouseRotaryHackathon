Submission Guidelines

Project Requirements

Submit a working application or prototype that addresses the hackathon theme. Projects can be web apps, mobile apps, desktop applications, or hardware projects.

Submission Format

Provide a GitHub repository link with your source code, a README file explaining your project, and a 5-minute demo video showcasing your solution.

Deadline

All submissions must be completed and submitted by October 26th, 2025 at 11:59 PM. Late submissions will not be accepted.

Team Rules

Maximum team size is 3 members. Solo participation is encouraged. All team members must be registered participants.
